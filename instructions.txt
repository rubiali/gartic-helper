🎯 EXTENSÃO SUGESTÕES DE PALAVRAS - INSTRUÇÕES DE INSTALAÇÃO

📁 PASSO 1: CRIAR PASTA
- Crie uma pasta chamada "extensao-sugestoes"
- Salve todos os 5 arquivos dentro desta pasta:
  * manifest.json
  * words.js
  * content.js
  * styles.css
  * popup.html

🔧 PASSO 2: INSTALAR NO CHROME
1. Abra o Google Chrome
2. Digite na barra: chrome://extensions/
3. Ative o "Modo do desenvolvedor" (toggle no canto direito)
4. Clique em "Carregar sem compactação"
5. Selecione a pasta "extensao-sugestoes"
6. A extensão aparecerá na lista

✅ PASSO 3: TESTAR
1. Vá para a página do jogo de palavras
2. O painel aparecerá automaticamente no canto direito
3. Conforme as dicas aparecem, as sugestões serão mostradas
4. Clique nas palavras para copiá-las

🎮 COMO FUNCIONA:
- Detecta automaticamente dicas no formato: CH___L___
- Busca na lista de palavras
- Mostra sugestões em tempo real
- Destaca letras já reveladas

📝 ADICIONAR PALAVRAS:
- Edite o arquivo words.js
- Adicione palavras na lista WORDS
- Recarregue a extensão

🚨 PROBLEMAS COMUNS:
- Se não aparecer: recarregue a página (F5)
- Se não detectar: verifique se o jogo tem a estrutura HTML correta
- Console (F12) mostra mensagens da extensão

🎯 ESTRUTURA DETECTADA:

  ...
</div>

📞 SUPORTE:
Se não funcionar, verifique:
1. Modo desenvolvedor ativo
2. Estrutura HTML do jogo
3. Console para erros (F12)

Divirta-se! 🎉
